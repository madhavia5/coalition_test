# 🏥 Patient Dashboard

A responsive web-based Patient Dashboard that displays patient health data, including vitals and medical insights, with interactive charts and downloadable reports.

## 🚀 Features

- 📊 Display patient details dynamically
- ❤️ Show vital signs:
  - Heart Rate
  - Temperature
  - Respiratory Rate
- 📈 Interactive Blood Pressure chart using Chart.js
- 📥 Download patient data as ZIP file
- 📱 Fully responsive design
- 🎨 Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js
- JSZip
- FileSaver.js
